# Placeholder Audio

This directory contains placeholder audio for development.
Replace these with final sound files when ready.

## Directory Structure

- `sfx/` - Sound effects (hits, UI sounds)
- `voice/` - Voice lines (excuse phrases)
- `music/` - Background music loops

## Audio Format

- Preferred: MP3 (with OGG fallback for compatibility)
- Sample rate: 44.1kHz
- See AssetManifest.js for duration guidelines

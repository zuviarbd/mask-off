# Placeholder Assets

This directory contains placeholder graphics for development.
Replace these with final assets when ready.

## Directory Structure

- `characters/` - Character sprite sheets
- `ui/` - UI elements (holes, stamps, backgrounds)
- `fx/` - Visual effects (particles, overlays)

## Sprite Sheet Format

- Characters: 200x250px per frame, 12 frames horizontal
- Boss: 250x300px per frame, 14 frames horizontal
- Effects: See AssetManifest.js for dimensions
